# 运动计划指南

## 有氧运动建议
- **频率**：每周 3~5 次，每次 30~60 分钟
- **强度**：保持在最大心率的 60%~75%
  - 公式：最大心率 = 220 - 年龄
  - 30岁人群减脂心率区间：114~143 bpm
- **推荐项目**：跑步、骑行、游泳、跳绳、椭圆机、爬楼梯
- **空腹有氧**：适合早起、无血糖问题的人群，可提升脂肪供能比例

## 无氧 / 力量训练
- **频率**：每周 2~3 次，每次 45~60 分钟，间隔至少 48 小时
- **重点肌群**：深蹲（腿臀）、硬拉（背腿）、卧推（胸）、引体（背）、肩推（肩）
- **每次训练覆盖 2~3 个大肌群**，小肌群作为补充
- **组间休息**：减脂期 30~60 秒（偏短利于燃脂）；增肌期 60~90 秒

## 最佳运动顺序（效率最高）
```
热身（5min）→ 无氧力量（45min）→ 有氧（20~30min）→ 拉伸（5~10min）
```

## 热量消耗参考（60kg成年人，30分钟）
| 运动项目 | 消耗热量 |
|---------|---------|
| 快走（6km/h）| 约 150 kcal |
| 慢跑（8km/h）| 约 300 kcal |
| 游泳（自由泳）| 约 350 kcal |
| 跳绳（中等强度）| 约 280 kcal |
| 动感单车 | 约 320 kcal |

## 注意事项
- 运动后补充蛋白质：30g 左右（乳清蛋白粉或 3~4 个鸡蛋清）
- 避免过度训练：同一部位至少休息 48 小时
- 体重基数大者（BMI > 28）：优先选择骑车、游泳，减少膝关节压力
- 运动前 30 分钟补充少量碳水（如半根香蕉）可提升运动表现
