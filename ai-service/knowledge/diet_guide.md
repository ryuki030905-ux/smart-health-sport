# 减脂饮食指南

## 核心原则
- **热量赤字**：每日摄入 < 消耗，才能减少体脂，建议每日缺口 300~500 kcal
- **优质蛋白**：每公斤体重摄入 1.2~1.6g 蛋白质（鸡胸、鱼、虾、豆类、蛋）
- **复合碳水**：优先选燕麦、红薯、糙米、全麦面包、玉米
- **蔬菜不限量**：西兰花、菠菜、生菜、黄瓜、番茄等低卡高纤蔬菜
- **健康脂肪**：坚果（每日 30g）、牛油果、橄榄油（每日 10~15ml）适量

## 三餐分配建议（减脂版）
- **早餐（30%）**：蛋白质 + 复合碳水
  - 推荐：2个水煮蛋 + 1碗燕麦粥 + 1个小苹果
- **午餐（40%）**：主食 + 蛋白 + 大量蔬菜
  - 推荐：糙米饭 150g + 香煎鸡胸 200g + 蒜蓉西兰花 300g
- **晚餐（30%）**：轻量碳水 + 蛋白
  - 推荐：红薯 150g + 清蒸三文鱼 150g + 凉拌菠菜
- **加餐**：无糖酸奶 150g 或坚果一小把（控制在 150kcal 以内）

## 常见误区
1. **不吃晚饭** → 基础代谢下降，容易反弹，肌肉流失
2. **只吃水果** → 果糖摄入过多，热量超标，不利于减脂
3. **完全断碳水** → 情绪低落、注意力下降、女性可能闭经
4. **迷信代餐/奶昔** → 营养单一，长期损伤基础代谢
